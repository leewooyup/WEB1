Preparing to become a web developer(What should I start with?)

Web development is divided into a front end and a back end.

Front-end is a development field that operates on a web browser, which implements everything users see and interact with on a website.
It mainly focuses on creating a user interface (UI) and a user experience (UX).
The languages used in front-end development include HTML, CSS, and Javascript. The technology used these days is ReactJS. It's a component-based open source library. It helps to create an excellent user interface.
(This technology has been developed and maintained by Facebook)

Backend is a development field that operates on servers.
Load/manage data and create APIs that can be utilized.
It is possible to smoothly perform tasks performed by users at the front end. Languages used in backend development include PHP, Javascript, java, C++, and Python.Among them, Node.js is a runtime environment that allows Javascript code to be executed outside the browser. Nodes are used to develop backend services such as mobile or web applications API.
